# Security Policy

## Supported Versions

This starter is intended to track the latest maintained base and integrated service templates.
Security fixes are applied to the latest release line.

| Version           | Supported |
| ----------------- | --------- |
| Latest release    | Yes       |
| Older releases    | No        |
| Unreleased `main` | No        |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Report suspected vulnerabilities privately via:

```text
okpainmoandrew@gmail.com
```

Please include a clear description, reproduction steps if available, affected service or root
workflow, and expected impact. We aim to acknowledge reports within 48 hours.
