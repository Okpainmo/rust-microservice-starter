# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standard

Participation should be professional, respectful, and focused on useful collaboration. Be kind when
disagreeing, assume good intent where reasonable, and keep feedback grounded in the work.

Unacceptable behavior includes harassment, personal attacks, publishing private information without
permission, or conduct that would reasonably be considered inappropriate in a professional setting.
